## 상황 설명 (응시자에게 주어진 정보)

QA 엔지니어가 토스 송금 기능을 수동 테스트하던 중 다음 현상을 발견했다.

- 1,500만원을 송금하려고 시도함
- 정책상 1,000만원 초과 송금은 **영상 통화 본인 인증** 단계가 떠야 함
- 그러나 비밀번호 입력 화면만 노출되고 그대로 송금이 진행됨
- 송금 처리 자체는 정상적으로 완료되어 수취 계좌에 입금까지 확인됨
- 동일 계정 / 동일 단말에서 3번 시도 모두 같은 현상 발생
- 다른 단말(iOS)에서 시도 시에는 정상적으로 영상 인증 화면이 노출됨
- 발생 단말: Android 14, 토스 앱 v6.123.0, 빌드 #45821
- 와이파이 / LTE 모두 동일하게 재현

## 작성 문제
위 상황을 표준 버그 리포트 양식에 따라 작성하시오.

- 재현 절차는 단계 분리 (선조건 / 단계 / 결과)
- 의심 원인과 영향 범위(영향 받는 사용자 추정치)도 작성
- 본 상황에 없는 정보를 임의로 추가하지 말 것 (필요하면 "확인 필요" 표기)
