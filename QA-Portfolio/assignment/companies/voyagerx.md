# 보이저엑스 (VoyagerX) · QA 채용 조사

> **조사일**: 2026-05-24

## 1. 회사 / 도메인 개요
- **회사 규모**: 약 59명 (소규모 AI 스타트업)
- **위치**: 서울 서초구
- **업력**: 9년
- **평균 연봉**: 5,307만원 (원티드 공시 기준)
- **주요 서비스**:
  - **Vrew** — AI 영상 편집기 (자동 자막 생성, AI 더빙)
  - **vFlat** — AI 모바일 스캐너
  - **온글잎** — AI 손글씨 폰트 생성기
- **특징**: B2C AI 데스크톱 SW + 모바일 앱. 글로벌 진출 활발

## 2. 채용 전형 절차

```
서류 (최대 1주일 내 결과) → 과제 전형 (메일 답변형, 사례비 지급) → 인터뷰
```

| 단계 | 비고 |
|---|---|
| 서류 | 이력서 + 자기소개서. 1주일 내 결과 통보 |
| 과제 | **별도 전달되는 질문에 답변 작성 → 메일 제출** |
| 사례비 | **과제 전형 참여 시 사례비 지급** (드문 케이스, 좋은 신호) |
| 인터뷰 | 과제 발표 + 직무 인터뷰 |

## 3. 과제 형식 (확인된 사실)

- **메일 답변형 과제**: 코드 구현보다 **문서 작성·답변** 비중 큼
- 사례비 지급은 회사가 지원자의 시간을 존중한다는 신호
- 과제 내용은 직무에 맞춰 개별 전달 (TC 작성, 테스트 전략, 결함 분석 등 추정)

## 4. 주요 업무 (QA 담당자)
- 신규 버전 릴리즈 **전** 품질 문제 검사
- 릴리즈된 버전의 품질 문제 찾기
- 품질 문제 **재현** 및 공유
- 개발 / 기획 / 디자인 등과의 협업을 통한 품질 관리

## 5. 준비 포인트

- **문서력이 핵심**: 메일 답변형이라 글 잘 쓰는 것이 변별 포인트. 결함 재현 절차, 영향 범위, 의심 원인을 명확한 문장으로
- **데스크톱 SW 특수성 이해**:
  - 로컬 파일 접근, 저장되지 않은 변경, 오프라인 동작
  - OS별 빌드 (Windows / macOS), 디스크 공간, 권한
- **Vrew 도메인 이해**: 자동 자막 → 편집 → 내보내기 흐름. 실제로 써보고 결함 잡아보는 게 가장 좋은 준비
- **AI 서비스 특유 이슈**: STT 정확도, GPU 처리 실패, 모델 한도, 처리 시간
- **소규모 회사 특성**: 한 명이 여러 영역 커버. T자형 역량 어필
