# 넥슨네트웍스 (게임 QA) · QA 채용 조사

> **조사일**: 2026-05-24
> **특성**: 일반 IT QA가 아닌 **게임 QA** 도메인. 진입 방식 / 평가 기준이 다름.

## 1. 회사 / 도메인 개요
- **법인**: 넥슨네트웍스 (넥슨코리아의 게임 QA 전문 자회사)
- **도메인**: 게임 (메이플스토리, 던전앤파이터, 카트라이더, FIFA 온라인 등)
- **위치**: 판교
- **QA 영역**: 게임 콘텐츠 QA + 운영 QA + 자동화

## 2. 채용 전형 절차

```
서류 → 온라인 인성검사 → 실무진 면접 → 인턴십 (6개월) → 전환 평가
```

> **채용연계형 인턴이 주된 진입로**. 신입~주니어는 인턴십 거치는 게 일반적.

| 단계 | 비고 |
|---|---|
| 서류 | 자기소개서 비중 큼 |
| 온라인 인성검사 | 별도 사전과제 없음 |
| 실무진 면접 | 자기소개 / 게임 경험 / 사고력 |
| 인턴십 | **6개월** 채용연계형 |
| 전환 평가 | 인턴 → 정규직 |

## 3. 게임 QA 특수성

### 일반 QA와의 차이
- **방대한 콘텐츠 vs 한정된 자원**: 신규 패치 / 이벤트 / 콘텐츠가 끊임없이 추가
- **리스크 기반 테스트** 필수: 어디서 결함이 가장 큰 임팩트를 만들지 판단
- **유저 리포트 분석**: 개발자 시선 + 유저 시선 모두 필요
- **데일리 스크럼**: 이슈 / 진행 상황 / 차단점 매일 공유

### 평가 키워드
- **리스크 기반 우선순위** 판단력
- **한정된 시간 / 인원 내 의사결정** 능력
- **게임에 대한 진정성** (자기소개서 + 면접)

## 4. 준비 포인트

- **게임 깊이 있는 플레이**: 지원하는 게임을 실제로 깊이 있게 해본 경험
- **유저 리포트 / 패치노트 분석**: 회사 운영 게임의 패치노트, 유저 컴플레인 흐름 분석
- **리스크 우선순위 사고**: "시간이 1주일밖에 없다면 무엇부터 테스트할 것인가" 같은 질문에 답할 준비
- **6개월 인턴 후 전환 = 장기전**: 인턴 자체가 본 채용. 6개월간 평가 받는다는 마인드
- **자기소개서 비중 큼**: 게임 경험 / 분석력 / QA에 대한 이해를 자소서에 풍부하게
