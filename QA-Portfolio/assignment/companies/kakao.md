# 카카오 (Kakao) · QA 채용 조사

> **조사일**: 2026-05-24
> **주의**: QA 직군 특화 후기 부족. 카카오 본사보다 **카카오페이, 카카오뱅크, 카카오엔터프라이즈** 등 자회사에 QA 채용이 더 많음.

## 1. 회사 / 도메인 개요
- **본사 도메인**: 메신저(카카오톡), 콘텐츠(웹툰/페이지), 카카오T, 카카오쇼핑 등
- **자회사**: 카카오페이, 카카오뱅크, 카카오엔터테인먼트, 카카오게임즈, 카카오엔터프라이즈, 카카오모빌리티 등 (각각 별도 채용)
- **QA 직군은 자회사가 활발**: 카카오페이 QA 자동화 엔지니어 등 (별도 문서 [kakaopay.md](kakaopay.md))

## 2. 채용 전형 절차

```
서류 → 코딩테스트 → 1차 면접 (사전과제 발표) → 인성 검사 → 2차 면접 → 합격
```

> 카카오 본사는 블라인드 채용. 어떠한 추가 서류 없이 바로 코딩테스트 응시 가능.

## 3. 과제 형식

### 카카오 본사 사전과제
- **기간**: 1주일
- **제출 형식**: 사전과제 풀이 + **발표 자료(PPT)** + 소스코드
- **1차 면접**: 사전과제 발표 → 사용 기술 / 엣지 케이스 / 트레이드오프 질문
- **변별력**: "사전과제 퀄리티가 부족해도 1차 인터뷰를 못 보는 경우는 없다" → 결과보다 **풀이 과정과 사고력** 평가

### 코딩테스트 특성
- **국내 IT 공채 중 가장 어렵다**고 평가
- 프로그래머스 플랫폼
- 매년 카카오 블라인드 채용 코딩테스트 기출 공개

## 4. 준비 포인트

- **발표 자료 준비**: 사전과제 풀이를 1차 면접에서 설명할 수 있어야 함 → "왜 이렇게 풀었는가" 정리 필수
- **엣지 케이스 의식**: 단순 동작뿐 아니라 예외 케이스 처리까지 사고
- **코딩테스트 난이도**: 카카오 본사 지원이면 알고리즘 별도 준비 필요
- **자회사 분리 지원**: QA라면 카카오페이 / 카카오뱅크 직접 지원 권장
