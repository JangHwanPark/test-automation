## 안내
다음 요구사항을 읽고, 아래 5가지 기준에 따라 요구사항을 분석하시오.

## 기능명
이메일 기반 회원가입

## 요구사항
- 사용자는 이메일, 비밀번호, 이름을 입력해 회원가입할 수 있다.
- 이메일은 형식이 올바른 경우에만 가입을 진행할 수 있다.
- 비밀번호는 8자 이상이어야 한다.
- 이미 가입된 이메일로는 다시 가입할 수 없다.
- 가입이 완료되면 입력한 이메일로 환영 메일이 발송된다.
- 가입이 완료되면 사용자는 자동으로 로그인된 상태가 된다.

## 작성 문제
아래 5가지 기준으로 요구사항을 분석하시오.
1. 사용자
2. 주요 기능
3. 상태 변화
4. 예외 상황
5. 외부 시스템 또는 정책 조건

## 작성 조건
- 각 항목은 1개 이상 작성할 것
- 상태 변화는 “A → B” 형태로 작성할 것
- 해당 항목이 정말 없다면 “없음”으로 적어도 됨
