# 무신사 (MUSINSA) · QA 채용 조사

> **조사일**: 2026-05-24

## 1. 회사 / 도메인 개요
- **도메인**: 패션 커머스, 무신사 스탠다드, 무신사 페이먼츠
- **계열**: 컬리페이, 무신사 등 그룹 내 다양한 채용
- **QA 직군**: QA Engineer

## 2. 채용 전형 절차

```
서류 → 1차 인터뷰 → 2차 인터뷰 → 처우 협의 → 최종 합격
```

> **사전 과제 / 전화 인터뷰**는 옵션으로 추가될 수 있음.

## 3. 준비 포인트

- **이력서 = 프로젝트 이해도**: 1차 면접에서 이력서 기반으로 **심도 있는 질문**. 본인이 한 프로젝트 모든 디테일 숙지
- **무신사 테크 미디엄 학습**: 회사 공식 블로그에서 QA 업무 / 자동화 / 품질 관리 글 다수 발행 — 면접 전 필독
- **QA의 정의 명확화**: 미디엄 글에 따르면 "QA는 요구사항·명세에 따라 기능·비기능 테스트 수행, 오류·결함·누락 검출" 로 정의 — 본인 관점도 정리
