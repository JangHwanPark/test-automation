# 당근 (Daangn / 당근마켓) · QA 채용 조사

> **조사일**: 2026-05-24

## 1. 회사 / 도메인 개요
- **도메인**: 동네 중고거래, 동네생활, 당근알바, 당근비즈프로필
- **법인**: 당근마켓 (서비스명은 "당근"으로 변경)
- **특성**: 컬처핏 강조 ("일 온도" 개념)

## 2. 채용 전형 절차

```
서류 → 화상 인터뷰 → 과제 전형 → 직무 인터뷰 → 컬처핏 인터뷰
```

> 포지션별로 절차가 조금씩 다름. **면접비 지급** 특이.

| 단계 | 비고 |
|---|---|
| 서류 | 이력서 / 자기소개서 |
| 화상 인터뷰 | 초반 스크리닝 |
| 과제 전형 | **5~7일** 안에 작성 가능한 분량, 직무 능력과 연관 |
| 직무 인터뷰 | 과제 발표 + 기술 질문 |
| 컬처핏 | "일 온도" — 당근의 일하는 방식 적합도 |

## 3. 과제 형식

- **기간**: 5~7일
- **성격**: **직무 능력과 연관된 내용**
- **분량**: 일주일 안에 충분히 가능한 양
- QA 직군 특화 과제 형식은 공개 자료 없음 — TC 작성 / 테스트 전략 / 자동화 코드 등 추정

## 4. 준비 포인트

- **5~7일 과제 = 시간 충분**: 시간이 부족해 못 했다는 변명 불가. 깊이 있는 풀이 + 회고
- **컬처핏 비중 큼**: 당근의 "일 온도", 동네 / 지역 도메인에 대한 진정성 어필
- **면접비 지급**: 회사가 지원자 시간을 존중 = 본인도 진지하게 준비해서 보답
- **지역 도메인 이해**: 위치 기반 매칭, 동네 한정 거래, 사기 / 신뢰 / 매너 온도 시스템
