# 우아한형제들 (배달의민족) · QA 채용 조사

> **조사일**: 2026-05-24

## 1. 회사 / 도메인 개요
- **도메인**: 배달 (배달의민족 앱), 라이더 앱, 사장님 광장, B마트
- **법인**: 우아한형제들
- **QA 직군**: **QA Engineer** + **QA(Test 자동화)** 별도 채용

## 2. 채용 전형 절차

```
서류 → 코딩테스트 → 기술 면접 (1·2차)
```

> **인적성 검사 없음**. 절차가 단순한 편.

| 단계 | 비고 |
|---|---|
| 서류 | 이력서 / 자기소개서 |
| 코딩테스트 | 입사 지원서 제출 후 메일로 전달 |
| 기술 면접 | 코딩테스트 제출 후 진행 |

## 3. 모집 조건

| 직군 | 자격 |
|---|---|
| QA Engineer | **경력 1년 이상**, SW 테스팅 기본 지식 |
| QA (Test 자동화) | **자동화 실무 경험 3년 이상** |

### 주요 업무
- 다양한 테스트 기법 / 툴을 활용한 **기능·비기능 테스트**
- 기획 / 디자인 / 개발 문서에 따른 **테스트 시나리오 작성**
- 테스트 설계 및 수행
- **결함 관리 및 산출물 작성**

## 4. 준비 포인트

- **코딩테스트 통과 = 필수 관문**: 인적성 없으니 코테가 1차 변별점. QA 직군이라도 알고리즘 기본기 필요
- **기술 면접에서 테스트 설계 깊이**: 단순 TC 작성이 아니라 **테스트 기법** (동등 분할, 경계값, 상태전이, 결정 테이블)을 명시적으로 활용한 경험 어필
- **자동화 직군 별도**: 자동화 3년 경력이면 자동화 직군으로 따로 지원 가능
- **배민 도메인**: 주문 / 결제 / 배달 추적 / 리뷰 같은 풍부한 상태 흐름
