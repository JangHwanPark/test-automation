# 토스 (Toss / 비바리퍼블리카) · QA 채용 조사

> **조사일**: 2026-05-24
> **주의**: QA 직군 특화 후기 공개 자료가 부족해, **개발직 후기에서 일관되게 확인되는 패턴 기반 추정** 포함.

## 1. 회사 / 도메인 개요
- **법인**: 비바리퍼블리카 (Viva Republica)
- **도메인**: 핀테크 (송금, 간편결제, 토스증권, 토스인슈어런스, 토스뱅크)
- **서비스 채널**: App (메인) + Web
- **계열사**: 토스증권, 토스뱅크, 토스플레이스, 토스페이먼츠 등 각각 별도 채용

## 2. 채용 전형 절차

```
지원 → 서류 → 코딩테스트/과제 → 직무 인터뷰 → 컬처핏 → 레퍼런스 체크 → 처우 협의 → 입사
```

| 단계 | 비고 |
|---|---|
| 서류 | 이력서. 토스는 자기소개서 비중 낮음 |
| 코딩테스트 / 과제 | 직군에 따라 코테 또는 **48시간(이틀) 구현 과제** |
| 직무 인터뷰 | 과제 코드 상세 리뷰 + 이력서 질문 |
| 컬처핏 인터뷰 | 토스 8가지 가치를 직접 묻지는 않음 |
| 레퍼런스 체크 | 이전 직장 동료 추천 확인 |
| 처우 협의 | 연봉 협상 |

> **진행 속도가 매우 빠름**. 코딩테스트 결과 발표 후 3일 내 이력서 제출, 1주일 내 기술 면접 일정.

## 3. 과제 / 사전 테스트 정보

### 과제 형식
- 개발직 후기 종합: **48시간(이틀) 동안** 주어진 요구사항을 구현
- 요구사항은 "프로젝트 하며 한번쯤 해볼만한 기능 수준"
- 표면은 간단해 보이지만 **물어볼 포인트가 많음** (엣지 케이스, 트레이드오프)
- QA 직군 과제 형식은 공개 자료 없음 — 자동화 코드 작성 + 테스트 전략 제출 형태 추정

### 면접 강조점
- 과제 코드에 대한 **상세한 코드 리뷰** 진행
- 모르는 부분은 **솔직히 인정** 후 아는 만큼 설명하는 태도 평가
- 토스 가치 직접 질문은 없음. 컬처핏은 "정말 열심히 사는 사람이면 답할 수 있는" 질문

## 4. 준비 포인트

- **48시간 과제 = 깊이 우선**: 표면 구현만 하면 인터뷰에서 깨짐. 엣지 케이스 / 트레이드오프를 미리 정리
- **코드 리뷰 대비**: 과제 코드의 모든 선택지를 설명 가능해야 함 ("왜 이 라이브러리?", "왜 이 패턴?")
- **컬처핏 = 토스 가치 학습**: 8가지 가치 직접 질문은 없지만, 답변 톤이 가치에 부합해야 함
- **속도**: 모든 단계가 1~3일 단위로 진행. 연락 늦으면 다음 단계 지연
