## 일반 트렌드 정리

## 활용 방법
1. 지원사 문서를 읽고 채용 절차 / 과제 유형 파악
2. **준비 포인트** 섹션의 항목들을 본 레포 `assignment/` 의 어느 카테고리로 연습할지 매핑
3. 회사 도메인에 맞는 실전 문제(`exam02` 크림 / `exam03~04` 토스 / `exam05` 보이저엑스)부터 풀이
4. 면접 직전 다시 읽고 회사별 특이사항(컬처핏, 진행 속도, 인터뷰 방식) 확인
